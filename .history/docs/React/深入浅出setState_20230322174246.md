在认识 setState 前，我们问几个常见问题
- `setState` 是同步还是异步？
- 如果是`异步`，怎么让它`同步`？
- 为什么要这样设计？

```js
const DemoState = (props) => {
  let [number, setNumber] = useState(0);

  const add = () => {
    setNumber(number+1);
    console.log(number);

    setNumber(number+1);
    console.log(number);
  }
  
  return (<div>
      <span>{ number }</span>
      <button onClick={() => { add() }} >点击加 1</button>
  </div>)
}
```
